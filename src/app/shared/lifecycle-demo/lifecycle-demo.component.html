    ngOnInit
    • ngOnInit() → Chạy 1 lần khi component khởi tạo.
    • ngOnInit() → Load API, set giá trị mặc định.

    ngOnChanges()
    • ngOnChanges() → Theo dõi dữ liệu Input() thay đổi.
    • ngOnChanges() → Khi Input() thay đổi.

    ngDoCheck()
    • gDoCheck() → Theo dõi thay đổi tùy chỉnh.
    • ngDoCheck() → Kiểm tra thay đổi tùy chỉnh.

    ngAfterContentInit()

    • Khi nội dung thẻ ng-content > được khởi tạo
    • Xử lý nội dung truyền vào từ parent

    ngAfterContentChecked()

    • Khi Angular kiểm tra nội dung thẻ ng-content>
    • Theo dõi thay đổi trong nội dung

    ngAfterViewInit()
    • ngAfterViewInit() → Dùng để thao tác với DOM.
    • ngAfterViewInit() → Chạy code sau khi View đã render.

    ngAfterViewChecked()

    • Khi Angular kiểm tra thay đổi trong view
    • Theo dõi thay đổi trong view/template
